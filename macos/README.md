# Mad for Audio — macOS 메뉴바 앱

메뉴바에 상주하는 하이파이 랙. 팝오버를 닫아도 라디오는 계속 재생됩니다.

## 빌드 & 설치

```bash
cd macos
./build.sh
cp -R "Mad for Audio.app" /Applications/   # 권장
open "/Applications/Mad for Audio.app"
```

Xcode Command Line Tools만 있으면 됩니다 (Xcode 프로젝트 불필요).

## 사용법

| 조작 | 동작 |
|---|---|
| 메뉴바 📻 아이콘 **좌클릭** | 랙 열기/닫기 (닫아도 소리는 계속) |
| 메뉴바 아이콘 **우클릭** | 메뉴 — 새로고침 · 브라우저에서 열기 · 미니 위젯 모드 · 로그인 시 자동 시작 · 종료 |
| 미니 위젯 모드 | 팝오버가 컴팩트 튜너 위젯(500×360)으로 전환 |
| 로그인 시 자동 시작 | 체크하면 맥 시작 시 메뉴바에 자동 상주 |

## 참고

- 웹뷰는 Safari 엔진(WebKit)이라 재생이 전부 네이티브 직결입니다. WebKit의
  MediaElementSource 제약으로 EQ·앰프 음색과 신호 반응 계기(파워미터 등)는 동작하지
  않고 조작·표시·저장만 됩니다. 음색·미터까지 원하면 Electron 메뉴바 앱(`tray/`,
  `npm run dist:mac`)을 사용하세요 — 이 앱은 가벼운 장시간 청취용으로 공존합니다.
- 설명서·미니 플레이어 링크는 기본 브라우저로 열립니다 — 팝오버 안에서
  페이지를 떠나면 재생이 끊기기 때문입니다.
- 테이프 녹음물은 이 앱 전용 저장소(WebKit 데이터 스토어)에 보존됩니다.
- 종료: 우클릭 → "Mad for Audio 종료" (앱을 끄면 소리도 멈춥니다).
