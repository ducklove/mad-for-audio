# 공개 모델 정체성 — v136

이 문서는 현재 설정 화면에서 선택할 수 있는 하이파이 모델의 정체성과 구현 범위를 정리한다.
공개 카탈로그는 **선택 모델 20종**(튜너 3, 이퀄라이저 3, 앰프 5, 카세트 데크 5,
턴테이블 4)과 선택 항목이 없는 **단일 오디오 타이머 DT-540**으로 구성된다.

## 적용 원칙

1. **식별 가능한 얼굴** — 실물 공개 자료에서 확인되는 패널 비례, 재질, 계기와 대표 조작을
   우선한다. 작은 레전드보다 큰 구조와 조작 순서가 먼저 읽혀야 한다.
2. **실제로 동작하는 조작** — 설명에는 현재 배선된 스위치, 노브, 미터와 음향 처리만 쓴다.
   장식으로 남은 요소를 기능처럼 설명하지 않는다.
3. **모델을 고를 이유** — 같은 종류 안에서도 시각 시그니처와 최소 하나 이상의 조작 또는
   음향 차이를 유지한다.
4. **절제된 효과** — 금속 반사, 유리 깊이, 조명 번짐과 질감은 구조를 보조하는 범위에서 쓴다.
   글로우가 글자나 눈금을 덮지 않게 한다.
5. **플랫폼 차이 명시** — 기계·시각 동작은 공통이고, Web Audio 기반 보이싱은 Chromium
   계열에서 완전하게 동작한다. Safari 계열은 가능한 조작 결과를 원샷으로 반영한다.

## 튜너 3종

### Yamaha T-2 — 녹색 다이얼과 서보 감각

- 다이얼 드래그 중 바늘·노브·주파수 표시가 함께 움직이고, 놓으면 가까운 방송국에 잠긴다.
- SIGNAL QUALITY와 TUNING 미터는 웜 아이보리 문자판으로 다이얼의 녹색과 분리된다.
- 서보 튜닝은 드래그 중 바늘·노브·디지털 STATION 표시를 함께 움직이고, 릴리즈 뒤
  가까운 방송국에 안착시킨다.
- 음향은 공개 튜너 가운데 기준이 되는 중립 프로필이다.

### McIntosh MR78 — 블루 글라스와 선택도

- 블랙 글라스 뒤의 다이얼과 레전드는 부분 얼룩 없이 하나의 블루 라이트박스로 읽힌다.
- NORMAL·NARROW·SUPER NARROW 선택도는 인접 채널 억제, 고역 개방감과 약전계 히스의
  균형을 바꾼다.
- 좌측 FUNCTION 표시·MULTIPATH 미터와 SIGNAL/TUNING 미터, PANLOC 형상과 여섯 노브가
  계측기형 레이아웃을 완성한다.

### Marantz Model 10B — 샴페인 패널과 CRT

- 샴페인 금속, 웜 롱 다이얼, 세리프 이탤릭과 CRT 튜닝 스코프가 핵심 식별 요소다.
- CRT의 초록 트레이스는 튜닝 중 편차와 잠금 뒤 오디오 신호에 반응한다.
- 보이싱은 세 튜너 중 가장 따뜻하지만 차이는 작은 범위로 제한한다.

## 이퀄라이저 3종

### Yamaha GE-5 — 즉시 읽히는 10밴드

- 30Hz·60Hz·120Hz·250Hz·500Hz·1kHz·2kHz·4kHz·8kHz·16kHz의 열 슬라이더가
  ±10dB로 동작한다.
- 일정한 간격과 백색 인디케이터로 작은 화면에서도 커브가 먼저 읽힌다.
- DEFEAT로 현재 커브와 플랫을 즉시 비교하며 모델별 설정은 따로 저장한다.

### Sansui SE-9 — 기억하고 움직이는 EQ

- 80Hz·160Hz·315Hz·630Hz·1.25kHz·2.5kHz·5kHz·10kHz의 8밴드를 좌우 채널로 표시하고
  ±12dB 범위에서 함께 제어한다.
- 앱 프리셋 10종과 사용자 메모리 A–D를 제공한다.
- 프리셋을 고르면 슬라이더가 목표 커브로 이동하고, A/B는 직전 커브와 현재 커브를 비교한다.
- 라디오·음반·테이프 소스에 맞는 추천 키는 은은한 테두리로만 표시한다.

### Technics SH-8065 — 33밴드 ⅓옥타브 슬라이더 월

- 실물의 핵심은 실버 브러시드 알루미늄 전면과 좌(위)·우(아래) 2단으로 늘어선 채널당
  33밴드(16Hz–25kHz, ⅓옥타브) 슬라이더 벽이다. 촘촘한 슬라이더 자체가 시그니처이며,
  캡을 이으면 실시간 커브 라인이 그려진다.
- **RANGE ±12dB / ±3dB** 스위치로 조정 폭을 바꾼다 — ±3은 미세 보정용으로 눈금이 다시
  그려지고 현재 커브가 새 폭으로 클램프된다.
- **CHARACTERISTIC NORMAL / INVERSE** 스위치는 설정한 커브의 역응답을 건다(슬라이더 위치는
  그대로, 응답만 반전 — 녹음의 EQ를 점검·상쇄하는 실기 동작).
- **EQ / THROUGH**(디피트)와 **POWER**(회로 차단·패널 소등)를 갖추며, **INPUT 150mV/1V**는
  하드웨어 입력 감도 선택을 그대로 표기한 스위치다.
- 오디오는 다른 EQ와 같이 L/R 링크(밴드 게인 33개)로 동작하고, 두 단 슬라이더는 같은 커브를
  그린다. 스펙트럼 아날라이저는 실기에 없으므로 넣지 않는다(그 기능은 형제 모델 SH-8066).

## 앰프 5종

### McIntosh MC2105 — 블루 POWER OUTPUT 미터

- 듀얼 미터의 원호·눈금·바늘 중심과 `POWER OUTPUT` 레전드는 서로 겹치지 않는다.
- 블루 미터, 그린 로고와 블랙 글라스의 명도 관계를 유지해 랙에서 멀리서도 식별된다.
- 솔리드스테이트 기준의 높은 제동과 작은 저·고역 보정을 쓴다.

### 8B TRIBUTE — EL34와 바이어스 의식

- 특정 실물의 정확한 복제품이 아니라 EL34 푸시풀 시대의 설계 언어와 앱 기능을 결합한
  **창작 재해석**이다.
- 스모크 갤러리의 대형 EL34 네 발과 소형 드라이버관 네 발, 단계적 워밍업과 순환식
  VU/CH A/CH B 바이어스 미터가 선택 이유다.
- 중간 새그와 보컬 중심의 온기를 갖되 계수로 제한된 보이싱이다.

### 91E TRIBUTE — 300B 싱글엔디드

- 특정 실물의 정확한 복제품이 아니라 극장용 300B 싱글엔디드 문법과 앱의 단계식 조작을
  결합한 **창작 재해석**이다.
- 벌룬형 300B와 정류관의 지연 점등, 웜 단일 미터, 단계식 볼륨이 식별 요소다.
- 2차 배음과 낮은 댐핑을 이용한 가장 부드러운 프로필이다.

### E-303 TRIBUTE — 실기 전면을 따른 정밀 제어형 앰프

- 밝은 실버·옅은 샴페인 금속, 독립된 두 피크 미터, 좌측 SPEAKERS와 우측 MC LOAD/HEAD AMP
  소형 버튼군을 실기 전면의 큰 구획 순서에 맞춘다.
- 하단은 LOUDNESS·BASS·TREBLE, 작은 TONE/TAPE/MODE 버튼군, 대형 VOLUME과 얇은
  BALANCE 레버, 세로 INPUT 버튼 열로 구성한다. 앱 기능 때문에 부품 형상을 바꾸지 않는다.
- 거의 중립인 프로필에 저음량 등청감 보상을 선택적으로 더한다.

### McIntosh MA2375 — 현대 올튜브 플래그십

- 넓은 블랙글라스 상부, 듀얼 블루 미터, 중앙 표시창과 직선형 스테인리스 하부가 정면 구조다.
- KT88 네 발의 크롬 케이지, 널링 노브와 반사면은 조명을 받되 미터보다 앞서지 않는다.
- 5밴드 톤과 KT88/Unity Coupled 계열의 넓은 대역·높은 제동 프로필을 제공한다.

## 카세트 데크 5종

### Nakamichi Dragon — 정보 밀도와 NAAC

- 좌측 POWER/TIMER/EJECT 열과 방향·Auto Azimuth 표시가 있는 작은 카세트 도어,
  중앙 4자리 카운터·세로 L/R 미터, 4행 트랜스포트/캘리브레이션 매트릭스,
  우측 레벨 노브·필터 열을 실기와 같은 순서로 분리한다.
- 외부에서 가져온 테이프의 고역 감쇠를 NAAC이 보정하고, 오토 리버스는 반대 면 재생 또는
  공면 리피트로 동작한다.
- 테이프 위치, 양면과 녹음 구간은 브라우저 저장소에 유지된다.

### Revox B215 — 컴퓨터 컨트롤

- 오픈 트랜스포트와 그린 표시, 모듈 키패드가 스튜디오 기기 같은 얼굴을 만든다.
- MEM은 위치를 기억하고 CUE는 그 위치로 감으며, RST는 0 위치, AUTO는 테이프별
  캘리브레이션을 수행한다.

### Tandberg TCD 3014A — 웜 미터와 헤드룸

- 우드 케이스, 큰 물리 노브와 트윈 아날로그 미터로 다른 검은 데크와 구분된다.
- 공개 데크 중 큰 수동 녹음 신호에서 포화를 가장 적게 적용하는 헤드룸 프로필이다.

### Pioneer CT-F1250 — 실버 시대 플루로스캔

- 월넛 케이스, 헤어라인 실버, 중앙 플렉시 도어와 청록 플루로스캔 미터가 핵심이다.
- 블루 카운터와 묵직한 60배속 와인딩으로 느린 기계적 성격을 유지한다.

### TEAC W-990RX — 독립된 두 웰

- 좌우 문 프레임, 카세트 중심, 릴 투명창과 상태 바를 같은 기준선에 맞춘다.
- 긴 릴 창이 양쪽 허브와 테이프 팩을 충분히 보여 주고, 중앙 제어부는 별도 모듈로 분리된다.
- A웰 재생과 B웰 녹음이 독립되어 예약 녹음 중에도 A웰을 사용할 수 있다.
- A→B 더빙, 고속 더빙과 연속 재생을 지원한다.

## 턴테이블 4종

### Technics SL-1200MK2 — 쿼츠 다이렉트 드라이브

- 피치 페이더는 ±8%로 동작하고 쿼츠 록은 0으로 복귀한다.
- 플래터 스트로브 도트는 정속에서 멈춰 보이며 피치 변화에 따라 흐른다.
- 네 모델 중 기동이 가장 빠르고 와우와 표면 노이즈가 낮다.

### Thorens TD 124 — 네 가지 속도

- 16⅔·33⅓·45·78 RPM을 순환하는 4단 속도 노브가 실제 재생 속도에 반영된다.
- 아이보리 모터보드, 월넛 플린스와 긴 관성으로 고전 아이들러의 성격을 만든다.

### Garrard 301 — 방송국형 레버

- 속도 미세 조정은 ±3% 범위에서 동작하고 BRAKE는 누르는 동안 플래터를 즉시 세운다.
- 크림 해머톤과 묵직한 우드 플린스, 두 레버의 위치가 핵심 식별 요소다.

### Linn Sondek LP12 — 절제된 벨트 드라이브

## 유닛 전원 위계 (v136 · v139에서 부팅 정책 확정)

실물 오디오처럼 기기를 각각 켜고 끈다. "시스템 재생/정지"라는 가상 전원은 폐기했다.
**기동은 항상 전체 통전** — 파워스트립에 물린 실기처럼 켜진 채 시작하고, 전원 상태는
세션 한정이다(영속하지 않음). **앰프는 명시 조작 전용**: 어떤 소스 조작·리모컨 경로도
앰프를 자동 점화하지 않는다(v138의 자동 점화가 만든 튜너/턴테이블 비일관을 제거).

```
벽전원 ─ DT-540 (항상 통전 · SWITCHED OUTLETS가 예약 시 튜너·데크를 임시 통전)
   ├─ 앰프  = 마스터: 꺼지면 스피커 무음. 소스·수신·녹음 신호는 실물처럼 계속 흐른다
   ├─ 튜너  = 수신 전원: ON이면 직전 채널(첫 설치는 93.1 KBS 1FM)에 잠긴다
   ├─ EQ    = 자체 전원 유지 (OFF = 평탄 통과)
   ├─ 데크  = 트랜스포트 관문: OFF면 PLAY/REC/와인딩이 거부된다
   └─ 턴테이블 = 자체 전원 유지 (기존 phonoPower)
```

- **전원 위치는 실물 근거를 따른다** — T-2: 좌측 첫 스위치. MR78: 별도 스위치 없음,
  **VOLUME 노브 통합**(반시계 끝 = OFF; 1977 실기 리뷰 확인) — 이에 따라 채널 목록은
  좌측 PANLOC으로 이관(우측 PANLOC=몰입과 대칭인 앱 문법). 10B: 우측 **OFF·ON·DIM
  3위치 노브**, DIM은 패널 조명 감광(실물 기능 재현). 데크 5종은 각 실물의 전면 위치에
  그려진 POWER에 bbox 배선(Dragon 좌상단 키·W-990RX 좌열·CT-F1250 좌하단, B215·TCD3014는
  좌하단에 최소 변형으로 추가).
- **물리 vs 리모컨** — 전면 스위치·다이얼은 물리 조작: 꺼진 튜너의 다이얼은 기계적으로만
  움직이고 안착 채널을 기억해 뒀다가 POWER ON에 연결한다. 채널 목록·헤더·미디어세션·
  트레이 명령은 리모컨: **튜너만** 깨운다(앰프·데크·TT는 각자 스위치 전용).
- **소스 선택** — 오토 센싱(매체 우선권) 유지. 입력 셀렉터는 실물에 있는 인티앰프
  (E-303·91E)만 가지며, 파워앰프(MC2105·8B)는 실물대로 셀렉터가 없다.
- 꺼진 유닛은 패널이 어두워진다(밝기 0.55). 본체 랙 전용 문법 — 위젯·트레이 미니
  플레이어는 실용 도구로 즉시 재생을 유지한다.
