# 공개 모델 완성도 평가 — v127 검증 재시작

이 문서는 오디오 기기 자체의 우열이 아니라 앱에 구현된 SVG와 상호작용의 완성도를 기록한다.
v112의 일괄 점수표는 구현 의도와 정적 코드 확인을 실제 결과보다 높게 반영했고, 추천 세트의
실렌더 대조도 부족했다. 따라서 그 점수와 S등급 표시는 현재 판정으로 사용하지 않는다.

## 등급 확정 조건

숫자와 등급은 아래 네 단계를 모두 통과한 모델에만 붙인다.

1. 공개 전면 사진 또는 설명서와 구획·부품 종류·재질을 대조한다.
2. Chromium과 WebKit에서 같은 크기로 단품을 캡처해 겹침, 잘림, 필터 누락을 확인한다.
3. 실제 추천 랙 크기로 다시 캡처해 작은 레전드와 조작부가 무너지지 않는지 확인한다.
4. 작동 요소의 히트 영역, 상태 저장과 미터·릴 애니메이션을 브라우저 테스트로 확인한다.

등급은 **S 95–100**, **A 90–94**, **B 85–89**, **C 80–84**, **D 79 이하**다.
S는 단순히 보기 좋은 모델이 아니라 실물 정체성, 세부 재질, 조명, 작동과 세트 조화에
뚜렷한 결함이 없을 때만 확정한다.

## 점수 구성

각 항목은 20점, 합계 100점이다.

- 구조·정렬 — 패널 비례, 기준선, 텍스트와 오브젝트 간격
- 실물 식별성 — 공개 자료에서 확인되는 고유 구획과 부품 문법
- 입체감·질감 — 금속, 유리, 목재, 노브, 카세트의 층과 반사
- 컬러·조명 — 색온도, 발광 균일성, 대비와 눈금 보존
- 기능·세부 — 실제 배선된 조작, 상태 피드백과 작은 표현의 일관성

## 나머지 공개 모델

나머지 모델은 삭제하거나 기존 점수를 자동 승계하지 않는다. 카탈로그에는 유지하되 같은
4단계 검증을 다시 거친 뒤에만 점수를 복구한다. 평균과 S등급 개수도 전 모델 재검증이 끝날
때까지 발표하지 않는다. 캡처 도구는 `tests/capture-svg-quality.js`와
`tests/capture-rack-quality.js`로 고정해 이후 평가가 같은 조건에서 반복되도록 한다.
