# 공개 모델 완성도 재평가 — v106

현재 공개 선택 모델 20종과 단일 오디오 타이머 DT-540을 같은 기준으로 다시 평가한 기록이다.
이 점수는 오디오 기기 자체의 우열이 아니라 **앱에 구현된 SVG와 상호작용의 완성도**를 뜻한다.
퇴역 모델은 평가 대상에서 제외한다.

## 평가 기준

각 모델은 아래 항목을 종합해 100점으로 평가했다.

- **구조·정렬** — 패널 비례, 기준선, 텍스트와 오브젝트의 간격, 작은 화면 판독성
- **실물 식별성** — 공개 자료에서 확인되는 재질, 계기, 조작부와 모델 고유 실루엣
- **입체감·질감** — 금속·유리·목재·관·카세트 셸의 층, 반사와 그림자
- **컬러·조명** — 색온도, 발광 균일성, 대비, 눈금과 레전드의 보존
- **기능·독창성** — 실제 배선된 고유 조작, 모델별 거동과 음향 차이
- **세부 표현** — 눈금·릴·노브·스위치·레전드와 상태 피드백의 일관성

등급은 **S 95–100**, **A 90–94**, **B 85–89**, **C 80–84**, **D 79 이하**로 나눈다.
소수점이 없는 모델 점수에 그대로 적용한다.

## 집계

- **총점:** 1,956 / 2,100
- **평균:** 93.14 → **93.1 (A)**
- **S등급 6종:** MR78, SE-9, QUAD 33/303, MA2375, Dragon, W-990RX
- **A등급 15종:** 나머지 공개 선택 모델 14종 + DT-540
- **B 이하:** 없음

v106 점수는 현재 공개 모델에 대한 이전 평가보다 낮추지 않고, 최근 정렬·질감·조명과 기능
개선분을 반영해 다시 고정한 값이다. 다만 실물과의 픽셀 단위 복제를 의미하지 않으며,
8B TRIBUTE·91E TRIBUTE·E-303 TRIBUTE는 창작 재해석이라는 성격 때문에 실물 유사성보다
내부 일관성과 기능 완성도를 중심으로 평가했다.

## 다음 개선 우선순위

1. 91E TRIBUTE — 작은 레전드와 조작부 기준선, 아이보리 금속의 미세 질감 보강
2. TCD 3014A — 미터 눈금·노브 주변 인쇄와 우드/금속 경계의 세부 밀도 향상
3. 8B TRIBUTE·E-303 TRIBUTE — 창작 모델임을 유지하면서 패널 정보 계층과 재질 차이를 정교화
4. T-2·GE-5·LP12 — 작은 화면에서 글자 대비와 단순 면의 미세 질감 보강

점수는 스크린샷만으로 확정하지 않는다. 데스크톱·모바일 렌더, 전원 ON/OFF, 재생·선국·
와인딩·프리셋 같은 상태 변화와 브라우저별 조작을 함께 확인한 뒤 갱신한다.
