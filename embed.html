위젯 · 임베드 안내

미니 플레이어를 iframe으로 내 사이트에 넣거나, 작은 팝업 창으로 띄우거나, postMessage로 다른 서비스와 연동할 수 있습니다.

미리보기

튜너형 (skin=tuner) — 빈티지 FM 튜너 페이스플레이트. 채널을 바꾸면 다이얼 바늘이 실제 주파수로 움직입니다.

튜너형 컴팩트 (300×150) — 크기를 줄이면 레이아웃이 자동으로 압축됩니다.

기본형 — 어디에나 어울리는 미니멀 플레이어.

위 위젯은 실제로 동작합니다. 채널을 고르고 재생해 보세요.

크기 가이드

위젯은 iframe 크기에 맞춰 스스로 압축됩니다. 아무 크기나 지정해도 되지만, 권장 조합은 다음과 같습니다.

크기스킨모습
460×280튜너형표준 — 다이얼·각인·스킨 전환 링크까지 전부 표시
300×150튜너형컴팩트 — 다이얼·주파수 표시창 유지, 각인·링크 숨김
380×200기본형표준
300×130기본형컴팩트 — 컨트롤 두 줄만
300×90튜너형/기본형초소형 — 다이얼 없이 컨트롤만

1. iframe으로 넣기

아래 코드를 복사해 원하는 페이지에 붙여넣으세요. allow="autoplay"를 주면 위젯 안의 재생 제어가 원활합니다.

빈티지 튜너 디자인으로 넣으려면 skin=tuner를 붙이고 높이를 280 정도로 잡으세요.

2. 작은 팝업 창으로 열기

버튼이나 링크에서 아래처럼 호출하면 미니 플레이어가 작은 별도 창으로 열립니다.

3. URL 파라미터

파라미터값설명
station채널 ID시작 채널을 지정합니다. 예: kbs1fm, kbs2fm, kbs1r, kbs2r, kbs3r, mbcsfm, mbcfm4u, cbsstd, cbsmusic, sbslove, sbspower, ebsfm, ytn, gugak, febc
autoplay1로드 즉시 재생을 시도합니다. 브라우저 정책상 차단되면 재생 버튼 대기 상태로 남습니다.
themedark(기본) / light밝은 배경의 사이트에 넣을 때 light를 사용하세요. (기본형 스킨에만 적용)
skintuner빈티지 FM 튜너 디자인 — 주파수 다이얼·바늘·LED 주파수 표시창·STEREO 램프가 있는 페이스플레이트로 표시됩니다. 권장 크기 460×280.

4. postMessage로 제어하기 (연동 API)

iframe이나 팝업으로 연 위젯에 메시지를 보내 제어하고, 위젯이 보내는 상태 이벤트를 받을 수 있습니다.

보낼 수 있는 명령

type추가 필드동작
fmRadio:playstation(선택)재생. station을 주면 해당 채널로 전환 후 재생
fmRadio:pause—일시정지
fmRadio:toggle—재생/일시정지 전환
fmRadio:setStationstation, autoplay(불리언)채널 전환. autoplay: true면 바로 재생
fmRadio:setVolumevalue (0–100)볼륨 설정
fmRadio:getState—현재 상태를 fmRadio:state로 회신

위젯이 보내는 이벤트 — fmRadio:ready(로드 완료), fmRadio:state(상태 변경·회신). 두 이벤트 모두 station, stationName, playing, loading, volume 필드를 담고 있습니다.

주의: 최초 재생은 브라우저 자동재생 정책 때문에 사용자의 클릭 등 제스처 안에서 명령을 보내야 확실하게 동작합니다.
